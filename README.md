# cmd4cs2

`sv_cheats 1` - включает читы на локальном сервере

`mp_warmup_end` - завершает разминку (`sv_cheats 1`)

`connect <ip>` - подключиться к серверу

`disconnect` - отключание от сервера

`noclip` - переключает режим свободного полёта (`sv_cheats 1`)

`bot_add_ct` - добавить бота за КТ сторону (`sv_cheats 1`)

`bot_add_t` - добавить бота за Т сторону (`sv_cheats 1`)

`bot_kick` - кикнуть ботов (`sv_cheats 1`)

`clear` - очищает консоль

`quit` - выйти из игры

`status` - показывает данные о сервере
